# Генератор варіантів

Веб-додаток для створення схем генерації, завантаження списків і швидкого отримання результатів. Усі дані зберігаються в localStorage.

## Швидкий старт

1. Відкрийте файл `index.html` у браузері.
2. Завантажте `.txt` списки через drag & drop або кнопку "Завантажити файл".
3. У блоці "Конструктор полів" додайте поля і налаштуйте типи.
4. Натисніть "Згенерувати".

## Формат `.txt` списків

- Один рядок = один варіант.
- Можливий підпункт через `;`.

Приклади:

```
Варіант
Варіант;Підпункт
```

## Схеми (інстанси)

- Можна створювати кілька схем, перемикатися між ними, клонувати, перейменовувати та видаляти.
- Кнопки керування схемами знаходяться у блоці "Конструктор полів".

## Типи полів

1. **Число** — генерує випадкове число між Мін і Макс.
2. **Строка** — випадковий елемент зі списку.
3. **Число + Строка** — випадкове число + випадковий елемент зі списку.
4. **Підпункт** — бере підпункт для вже згенерованого значення з попередніх полів.

## Форматування (MD)

У кожного поля є інпут "Формат рядка (MD)".

Плейсхолдери:
- `{label}` — назва поля
- `{value}` — значення
- `{index}` — номер варіанту

Приклади:

```
**{label}**: _{value}_
```

За замовчуванням використовується `{label}: {value}`.

## Генерація кількох варіантів

У блоці "Генерація" задайте кількість у полі "Кількість" (1–50) і натисніть "Згенерувати".

## Експорт та імпорт

- **Експорт даних** — зберігає всі списки, схеми та налаштування у файл `generator-data.json`.
- **Імпорт даних** — відновлює дані з JSON-файлу.
